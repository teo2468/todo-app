// ============================================ // ToDo アプリ GAS コード（全面書き直し版） // ============================================ // 設定: 以下の3つを自分の値に置き換えてください var SPREADSHEET_ID = '1lIYGcsu_XWzweXHj82M4QM2pCoSLtSiFsBj5hLfzNkg'; var LINE_TOKEN = 'JessPeHRBhbEDKt3GLAD6UQeG3QuV6QBQupkPIQKuQJM93vM9Z5d6X9QCtEUo3wJy2cJxoY7fjJxBbVBGwOnltL2QKdS4zJISJ68WLl2SSyz47x9XNieCYAqJHyf/2n7kP0+98HP3/d90nP0Jtg8OgdB04t89/1O/w1cDnyilFU='; var APP_URL = 'https://todo-app-tawny-iota-98.vercel.app'; // シート名 var SHEET_NOTIFICATIONS = 'notifications'; var SHEET_USERS = 'users'; var SHEET_SYNC = 'sync'; // ============================================ // doGet: データ読込（load）用 // アプリからGETリクエストで呼ばれる // ============================================ function doGet(e) { try { var action = e.parameter.action; var userId = e.parameter.userId; if (action === 'load' && userId) { return handleLoad(userId); } return jsonResponse({ error: 'invalid request' }); } catch (err) { return jsonResponse({ error: err.message }); } } // ============================================ // doPost: LINE Webhook + アプリからの書き込み // ============================================ function doPost(e) { try { var body = JSON.parse(e.postData.contents); // --- LINE Webhook（eventsプロパティあり）--- if (body.events) { body.events.forEach(function (event) { if (event.type === 'follow') { handleFollow(event); } }); return ContentService.createTextOutput('OK'); } // --- アプリからのリクエスト --- switch (body.action) { case 'notify': return handleNotify(body); case 'cancel': return handleCancel(body); case 'sync': return handleSync(body); default: return ContentService.createTextOutput('unknown action'); } } catch (err) { return ContentService.createTextOutput('error: ' + err.message); } } // ============================================ // LINE フォローイベント処理 // 友だち追加 → トークン生成 → リンク送信 // ============================================ function handleFollow(event) { var lineUserId = event.source.userId; var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var sheet = getOrCreateSheet(ss, SHEET_USERS, ['userId', 'lineUserId', 'createdAt']); var data = sheet.getDataRange().getValues(); // 既存ユーザーチェック（ブロック解除→再追加の場合） for (var i = 1; i < data.length; i++) { if (data[i][1] === lineUserId) { // 既にいる → 同じトークンのリンクを再送 sendWelcomeMessage(lineUserId, data[i][0]); return; } } // 新規ユーザー → トークン生成して保存 var token = generateToken(); sheet.appendRow([token, lineUserId, new Date().toISOString()]); sendWelcomeMessage(lineUserId, token); } // ============================================ // 通知登録 // ============================================ function handleNotify(body) { var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var sheet = getOrCreateSheet(ss, SHEET_NOTIFICATIONS, ['userId', 'taskId', 'taskName', 'notifyAt']); sheet.appendRow([body.userId, body.taskId, body.taskName, body.notifyAt]); return ContentService.createTextOutput('ok'); } // ============================================ // 通知キャンセル // ============================================ function handleCancel(body) { var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var sheet = ss.getSheetByName(SHEET_NOTIFICATIONS); if (!sheet) return ContentService.createTextOutput('ok'); var data = sheet.getDataRange().getValues(); // 下から削除（行番号がずれないように） for (var i = data.length - 1; i >= 1; i--) { if (data[i][0] === body.userId && data[i][1] === body.taskId) { sheet.deleteRow(i + 1); } } return ContentService.createTextOutput('ok'); } // ============================================ // データ同期（保存） // アプリ → GAS → syncシートに上書き保存 // ============================================ function handleSync(body) { var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var sheet = getOrCreateSheet(ss, SHEET_SYNC, ['userId', 'data', 'updatedAt']); var data = sheet.getDataRange().getValues(); // 既存行を探して上書き for (var i = 1; i < data.length; i++) { if (data[i][0] === body.userId) { sheet.getRange(i + 1, 2).setValue(body.data); sheet.getRange(i + 1, 3).setValue(body.updatedAt); return ContentService.createTextOutput('ok'); } } // 新規ユーザー → 行追加 sheet.appendRow([body.userId, body.data, body.updatedAt]); return ContentService.createTextOutput('ok'); } // ============================================ // データ読込 // GETリクエストで呼ばれ、JSONレスポンスを返す // ============================================ function handleLoad(userId) { var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var sheet = ss.getSheetByName(SHEET_SYNC); if (!sheet) { return jsonResponse({ data: null }); } var data = sheet.getDataRange().getValues(); for (var i = 1; i < data.length; i++) { if (data[i][0] === userId) { return jsonResponse({ data: data[i][1], updatedAt: data[i][2] }); } } return jsonResponse({ data: null }); } // ============================================ // 毎分トリガー: 通知チェック＆LINE送信 // ============================================ function checkAndNotify() { var ss = SpreadsheetApp.openById(SPREADSHEET_ID); var nSheet = ss.getSheetByName(SHEET_NOTIFICATIONS); if (!nSheet) return; var uSheet = ss.getSheetByName(SHEET_USERS); if (!uSheet) return; var now = new Date(); var notifications = nSheet.getDataRange().getValues(); var users = uSheet.getDataRange().getValues(); // userId → lineUserId のマップを作成 var userMap = {}; for (var j = 1; j < users.length; j++) { userMap[users[j][0]] = users[j][1]; } // 送信済み行を記録（後で削除） var rowsToDelete = []; for (var i = 1; i < notifications.length; i++) { var userId = notifications[i][0]; // col B = taskId, col C = taskName, col D = notifyAt var taskName = notifications[i][2]; var notifyAt = new Date(notifications[i][3]); if (notifyAt <= now) { var lineUserId = userMap[userId]; if (lineUserId) { sendLineMessage(lineUserId, '🔔 ' + taskName + ' の時間です！'); } rowsToDelete.push(i + 1); } } // 送信済み行を下から削除 for (var k = rowsToDelete.length - 1; k >= 0; k--) { nSheet.deleteRow(rowsToDelete[k]); } } // ============================================ // ユーティリティ関数 // ============================================ // トークン生成（10文字のランダム英数字） function generateToken() { var chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789'; var token = ''; for (var i = 0; i < 10; i++) { token += chars.charAt(Math.floor(Math.random() * chars.length)); } return token; } // ウェルカムメッセージ送信 function sendWelcomeMessage(lineUserId, token) { var url = APP_URL + '?user=' + token; var message = 'ToDoアプリへようこそ！\nこのリンクからアプリを開いてね：\n' + url; sendLineMessage(lineUserId, message); } // LINE メッセージ送信 function sendLineMessage(lineUserId, message) { UrlFetchApp.fetch('https://api.line.me/v2/bot/message/push', { method: 'post', contentType: 'application/json', headers: { 'Authorization': 'Bearer ' + LINE_TOKEN }, payload: JSON.stringify({ to: lineUserId, messages: [{ type: 'text', text: message }] }) }); } // シート取得（なければヘッダー付きで作成） function getOrCreateSheet(ss, name, headers) { var sheet = ss.getSheetByName(name); if (!sheet) { sheet = ss.insertSheet(name); sheet.appendRow(headers); } return sheet; } // JSONレスポンス作成 function jsonResponse(obj) { return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); }
